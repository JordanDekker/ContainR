Welcome to ConTaInR v1.0!

ConTaInR is a web-based tool for analysis of metagenomic data. It contains four main functionalities:
  1. The first one helps you to filter and preprocess your FastQ files.
  2. The second one is to BLAST (uses BLAST+) that reads against an inhouse reference database.
  3. The third one visualizes the BLAST results in a sunburst and graphs and gives the option to extract the desired results.
  4. The last one uses BLAST to annotate the sequences based on taxonomy. Several attributes can influence the quality of reads in metagenomic datasets.
To use these functionalities, ConTaInR uses FASTQ files as input. There are two forms of output:
  • A TSV file which contains the session information. In this file is shown which steps ConTaInR has taken and which filters it has applied. (Warning: please read the information on the bottom of the preprocess page before opening the file in Excel.)
  • A FASTA file with the sequences bound by the selection in the sunburst.
